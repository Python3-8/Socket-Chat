# Socket Chat
With these files running, two devices get an invisible connection in between them and can send and receive messages.
* Setup
  * Download the Chat Server and Client files.
  * Move the downloaded files to a new directory.
  * Add that directory to your **PATH**.

* The Chat App
   * Open up two command line windows.
   * In the first one, run the command "**_chat-server_**."
   * In the second command line window, run the command "**_chat-client_**."
   * Follow the instructions shown on both command line windows.

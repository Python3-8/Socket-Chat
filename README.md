# Socket Chat
With these files running, two devices get an invisible connection in between them and can send and receive messages.
1) Download the Chat Server and Client files.
2) Move the downloaded files to a new directory.
3) Add that directory to your PATH.
4) Open up two command line windows.
5) In the first one, run the command "chat-server."
6) In the second command line window, run the command "chat-client."
7) Follow the instructions shown on both command line windows.
